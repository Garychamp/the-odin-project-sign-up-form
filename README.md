# The Odin Project — Sign-Up Form

A sign-up form project built as part of [The Odin Project](https://www.theodinproject.com/) Intermediate HTML and CSS curriculum.

## Overview

This project focuses on creating a styled sign-up form using semantic HTML and custom CSS, applying key concepts including form validation styling, pseudo-classes, and layout techniques.

## Features

- **Two-panel layout** — a full-height decorative image panel on the left and a form panel on the right
- **Odin Project logo overlay** — logo centered over the background image on a semi-transparent banner
- **Background photo attribution** — "Photo by Halie West on Unsplash" displayed at the bottom of the image
- **Form fields** — First Name, Last Name, Email, Phone Number, Password, and Confirm Password
- **Input validation styling:**
  - Default inputs have a light border (`#E5E7EB`)
  - Password inputs display a red border when the value is invalid (`:invalid` pseudo-class)
  - Focused inputs display a blue border with a subtle box-shadow (`:focus` pseudo-class)
- **Form actions** — Create Account submit button and a Log in link for existing users

## Technologies Used

- HTML5
- CSS3

## Concepts Practiced

- CSS Flexbox for layout
- CSS pseudo-classes (`:focus`, `:invalid`)
- `position: absolute` / `position: relative` for image overlays
- `object-fit: cover` for background image handling
- Form structure and accessible labelling
